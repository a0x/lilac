golb
----

Not `gold` neither `glob`, `golb == reverse(blog)`

Simple static blog for programmers.

Usage
-----

```
Usage:
  golb <command>
  golb (-h | --help)
  golb (-v | --version)
Options:
  -h   --help    show this
  -v --version   show version
Commands:
  init        init current directory as a golb working directory
  build       build blog
  clean       rm output files
```
